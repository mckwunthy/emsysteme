########################## SYSTEME DE GESTION D'EVENEMENT PHP & MySQL  ################################

--Description : plateforme de gestion d'évènement

1- Création d'évènement :
	*Utilisateur : 	- connectez-vous à votre compte, dans l'onglet "manage event" remplisser le 
				formulaire de gauche pour créez un évènement
				
				- à droite, la liste de tous les évènements. Avec les bouttons "updat" ou "delete"
					modifiez ou supprimez les évènement que vous avez créé

2- Gestion des évènements et réservations  
	*Utilisateurs :
		- faite une demande de reservation (boutton "reserver")
		- les réservation validées apparraissent sur votre profil
		- si vous avez créé un évènement, dans l'onglet "booking", validez les demandes 
			de réservation en attente


3- Authentification :

	* créez un compte utilisateur
			cliquez sur "create account"
			renseignez votre email et mdp et validez
			completez vos informations sur votre profil

4- Recherche et filtrage
	*faites une recherche par "nom", "date (aaa-mm-jj)", "heure"
		dans l'onglet "event" à droite, remplissez le formulaire pour faire la recherche
		NB: vous pouvez renseigner uniquement les premiers mots

NB : emsysteme.sql = base de donnee


Bon usage ;-)
